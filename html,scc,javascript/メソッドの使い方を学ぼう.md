```
<?php

namespace Src\Station10;

class Question
{
    public function main(string $animal): string
    {
        return $this->getAnimalName($animal);
    }

    private function getAnimalName(string $animal): string
    {
        if($animal ==='猫'){
             return 'ミケ';
        }elseif($animal === '犬'){
             return 'ポチ';

        }else{
             return 'わかりません';
        };
       
    }
}

```

# mainメソッドについて説明する。

` main(string %animal)`についてだ。
この`string`というのは引数が文字列以外認めないという意味である。
そして、最後の`: string`は返り値が文字列以外認めないという意味である。
そして、`$this->getAnimalName($animal);`で`getAnimalName`メソッドに移動する。この時mainメソッドの引数が受け継がれている。
`getAnimalName`メソッドで処理が終わったら、再びmainメソッドに戻る。この時、`return (引数)`の形になっている。
